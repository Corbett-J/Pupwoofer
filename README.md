# Pupwoofer

Personal home server/discord bot with a variety of modules including running game servers, some web scraping, and permission based discord commands.

All work here is just intededed to be 'good enough' for the personal use of myself and some friends, so there are some issues here and there such as broken functionality for edge cases, unfinshed features on the main branch, some hard coded paths, missing types, etc.

If you're interested in a walkthrough of the code, feel welcome to reach out to me via LinkedIn!
